# 🦖 小恐龙 PK 小狮子 (Stick Strategy Game)

## 🎮 游戏简介
这是一个经典的**取火柴策略游戏**（Nim Game 变种）。你需要扮演聪明的**小恐龙**，与擅长逻辑计算的 AI **小狮子**对战。

## ✨ 特色功能
1.  **高度自定义**：
    *   🔥 **火柴总数**：10-100 根任选。
    *   ✋ **每次最多取**：1-10 根任选。
2.  **智能 AI**：小狮子会根据当前的规则动态调整策略（使用 Modulo 算法）。
3.  **精美 UI**：全中文界面，可爱的头像，流畅的动画效果。
4.  **单文件运行**：所有代码集成在 `index.html` 中，无需安装任何依赖，浏览器打开即玩。

## 🕹️ 游戏规则
1.  在这个游戏中，**拿走最后一根火柴的人获胜**。
2.  双方轮流取火柴。
3.  每人每次至少取 1 根，最多取 N 根（由你设置）。
4.  如果你能算出策略，你就能 100% 战胜小狮子！
    *   *💡 提示：试着让这轮剩下的火柴数保持是 `(每次最多取数 + 1)` 的倍数。*

## 🚀 如何开始
1.  直接用浏览器打开 `index.html` 即可。
2.  设置你喜欢的规则，点击【小恐龙 PK 小狮子】开始游戏！
